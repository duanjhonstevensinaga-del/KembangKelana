Cara Pasang QR Code Pembayaran
1
Buka aplikasi m-banking / dompet digital kamu, lalu generate QR Code untuk masing-masing nominal paket (Bronze Rp 5.000 · Silver Rp 10.000 · Gold Rp 15.000).
2
Screenshot QR Code tersebut, pastikan resolusi bagus dan mudah di-scan.
3
Upload ke kotak QR masing-masing paket di bawah ini — gambar akan otomatis disimpan ke folder qr/ dengan nama file yang sudah ditentukan.
4
Selesai! QR akan langsung muncul di modal pembayaran ketika user membeli paket.

📁 Cara Manual (via File Manager / FTP)
Alternatif jika upload lewat halaman ini tidak tersedia di hosting kamu.
  1. Buat folder qr/ di root folder project kamu (sejajar dengan index.html).
  2. Simpan gambar QR dengan nama tepat sesuai tabel berikut:
Paket Nama File Path Lengkap Nominal
🥉 Bronze qr-bronze.png qr/qr-bronze.png Rp 5.000
🥈 Silver qr-silver.png qr/qr-silver.png Rp 10.000
🥇 Gold qr-gold.png qr/qr-gold.png Rp 15.000
💡 Format file yang didukung: PNG (disarankan), JPG, atau WEBP. Pastikan ukuran file tidak lebih dari 2 MB dan QR terlihat jelas saat diperbesar.
